### Get started with JavaScript
1. Install nvm
   ```bash
   brew install nvm
   ```
2. Install node.js runtime
   ```bash
   nvm install 22.6.0
   nvm alias default 22.6.0
   ```

### Get started with Go
1. Install goenv
   ```bash
   brew install goenv
   ```
2. Install go runtime
   ```bash
   goenv install 1.23.0
   goenv global 1.23.0
   ```
